# Requirements

## Math Adventure - Kids Friendly Math Application

### System Requirements

- **OS**: macOS, Windows, Linux
- **Git**: Version 2.0 or higher (for version control)

### Browser Requirements

The application is built with vanilla HTML, CSS, and JavaScript and runs in all modern web browsers:

- **Chrome**: Version 90+
- **Firefox**: Version 88+
- **Safari**: Version 14+
- **Edge**: Version 90+

### No External Dependencies

This is a **zero-dependency** web application. It uses:
- Pure HTML5
- Vanilla CSS3 (with animations and gradients)
- Vanilla JavaScript (ES6+)

### File Structure Requirements

```
Math_app/
├── index.html              # Main HTML file
├── styles.css              # All styling and animations
├── script.js               # Game logic and interactions
├── Picture/                # Player images folder
│   ├── Nilan.jpeg         # Player 1 image
│   └── Mithula.jpeg       # Player 2 image
├── package.json            # Project metadata
├── requirements.txt        # This file
├── README.md              # Documentation
└── .github/
    └── workflows/
        └── deploy.yml     # GitHub Actions workflow
```

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Math_app.git
   cd Math_app
   ```

2. **Open the application:**
   - Simply open `index.html` in any modern web browser
   - No server setup required
   - Works completely offline

3. **Optional - Serve locally (for development):**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if installed)
   npx http-server
   
   # Using Node.js with Live Server
   npx live-server
   ```

### Deployment

The application is deployed via GitHub Actions to GitHub Pages automatically when code is pushed to the `main` branch.

**To enable deployment:**
1. Go to your GitHub repository
2. Settings → Pages
3. Select "GitHub Actions" as Source
4. Your app will be live at: `https://yourusername.github.io/Math_app/`

### Support & Troubleshooting

**Images not displaying:**
- Ensure `Picture/` folder contains `Nilan.jpeg` and `Mithula.jpeg`
- Images must be in JPEG format (not HEIF)
- Check file permissions

**Application not loading:**
- Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
- Try incognito/private mode
- Update browser to latest version

**Score not saving:**
- This is by design - scores reset on page refresh
- To persist scores, localStorage can be implemented
